\doxysubsection{Detailed Description}
Abstract base class for managing an attachment list (\mbox{\hyperlink{classGateListManager}{Gate\+List\+Manager}}) and for inserting new objects. 


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGateListMessenger}{Gate\+List\+Messenger}} -\/ by \href{mailto:Daniel.Strul@iphe.unil.ch}{\texttt{ Daniel.\+Strul@iphe.\+unil.\+ch}}
\item The \mbox{\hyperlink{classGateListMessenger}{Gate\+List\+Messenger}} inherits from the abilities/responsabilities of the \mbox{\hyperlink{classGateClockDependentMessenger}{Gate\+Clock\+Dependent\+Messenger}} base-\/class, i.\+e. the creation and management of a Gate UI directory for a Gate object
\item In addition, the main responsability of this messenger is to handle an attachment list, and to allow the insertion of new attachments.
\item It proposes and manages commands specific to the attachment lists definition of the name of a new attachment, listing of possible choices, list of already created attachments, and insertion of a new attachment
\item The class contains two pure virtual methods\+: Dump\+Map() and Do\+Insertion() These methods must be implemented in derived concrete classes. 
\end{DoxyItemize}
