\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classGateLinearRepeater}{Gate\+Linear\+Repeater}} models a repetition of an object along a line,. 

a pattern similar to the repetition of the scanner rings in a PET scanner


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGateLinearRepeater}{Gate\+Linear\+Repeater}} -\/ by \href{mailto:Daniel.Strul@iphe.unil.ch}{\texttt{ Daniel.\+Strul@iphe.\+unil.\+ch}}
\item The linear repeater uses three parameters\+:
\begin{DoxyItemize}
\item a number of repetitions
\end{DoxyItemize}
\item a repetition vector
\item a centering flag Based on these parameters, it repeats an object at regular steps in a line, the step size being given by the repeat vector. If the centering flag is on, the series of copies is centered on the volume current position. If this flag is off, the first copy in the series is located at the volume current position. 
\end{DoxyItemize}
