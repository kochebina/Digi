\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classGateAngularRepeater}{Gate\+Angular\+Repeater}} models a repetition of an object around an axis,. 

a pattern similar to the repetition of the scanner blocks in a cylindrical PET.

A Rsector can be shifted along Z each modulo N Rsector starting at a specified Rsector.

Each element of a group of \mbox{[}m\+\_\+modulo\+Number\mbox{]} is shifted according to its position inside a sequence (see \char`\"{}j\char`\"{} bellow).

Without specifing the shift(s) and the sequence length in the macro sequence, we get no shift and the standard ring repeater.

In a \mbox{[}m\+\_\+modulo\+Number\mbox{]} sequence, elements shift value is defined as \+:

Z shift(i) = Zshift\+\_\+j where j=(im\+\_\+modulo\+Number)+1 \+: position in a sequence and i position on the full ring.

See also \char`\"{}\+List Mode Format Implementation\+: Scanner geometry description Version 4.\+1 M.\+Krieguer \& al.\char`\"{} ,

in particular ASCII header file entrie \char`\"{}z shift sector \mbox{[}\+NIDShifted\mbox{]} mod \mbox{[}\+Nperiodicity\mbox{]}\+: \mbox{[}\+Zshift\mbox{]} \mbox{[}units\mbox{]}\char`\"{}

related \+: Gate\+To\+LMF class. Please take care that the present LMF format (22.\+10.\+03) support only for the moment a geometry with a cylindrical symetry. For example, a repeater starting at 0 degre and finishing at 90 degree (a quarter of ring) wont be supported by the LMF output.


\begin{DoxyItemize}
\item The angular repeater uses six parameters\+:
\item a number of repetitions
\item a start point and end-\/point of an axis
\item an autorotation flag
\item a start angle and an angular span
\item a repeated sequence length
\item the Z shift of element in the \char`\"{}modulo\char`\"{} sequence. Based on these parameters, it repeats an object at regular steps around an axis. If the autorotation flag is on, the orientation of the copies changes along with their position, as in a PET scanner geometry. If this flag is off, their orientation is the same for all copies (like in a fun-\/fair wheel). 
\end{DoxyItemize}
