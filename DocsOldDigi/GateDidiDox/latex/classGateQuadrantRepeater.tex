\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classGateQuadrantRepeater}{Gate\+Quadrant\+Repeater}} models a repetition of an object in a triangle-\/like. 

pattern similar to the one used in Derenzo phantoms (this repeater was

specifically designed to allow the modelling of such phantoms)


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGateQuadrantRepeater}{Gate\+Quadrant\+Repeater}} -\/ by \href{mailto:Daniel.Strul@iphe.unil.ch}{\texttt{ Daniel.\+Strul@iphe.\+unil.\+ch}} (Oct 2002)
\item The quadrant repeater uses three parameters\+:
\begin{DoxyItemize}
\item a number of lines
\end{DoxyItemize}
\item an orientation angle (in the XY plane)
\item a copy spacing Based on these parameters, it repeats an object in the XY plane in a series of copy lines. The orientation angle gives the direction of line-\/replication, each line being orthogonal to this direction. The number of copies increases from one line to the next one. The repetition vectors both between lines and between copies on a line are computed so that the copy spacing is identical between each copy and its nearest neighbours. The resulting pattern is an equilateral triangle.
\item A fourth optional paramter allows to reject positions beyond a limit 
\end{DoxyItemize}
