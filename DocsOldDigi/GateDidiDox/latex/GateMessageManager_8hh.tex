\doxysubsection{Detailed Description}
Class \mbox{\hyperlink{classGateMessageManager}{Gate\+Message\+Manager}} and Macros for outputing messages in Gate. 

There are 4 kinds of messages \+:
\begin{DoxyItemize}
\item Messages (normal messages)
\item Debug messages (not compiled in release)
\item Warnings
\item Errors
\end{DoxyItemize}

There are also \char`\"{}types\char`\"{} of messages which are strings which identify the nature of the message (for example \+: \char`\"{}\+Core\char`\"{} messages are generated by the core classes of the library, there can be a type of message for each type of Node, and so on...)

A type of message must be declared by registering it into the Message\+Manager. This is done by a line like \+: Gate\+::\+Message\+Manager\+::\+Register\+Message\+Type(\char`\"{}\+Core\char`\"{},\char`\"{}\+Messages generated by the core classes of the library\char`\"{},5); where \+:

-\/The first string is the type of the message (the keyword which will be used to generate a message of this type) -\/The second string is help string -\/The integer is the initial level for the messages of this type (see below).

To generate a message of a known type then use one of the macros \+: Gate\+Message, Gate\+Debug\+Message, Gate\+Warning, Gate\+Error or their variants.

example \+:

Gate\+Message(\char`\"{}\+Core\char`\"{},4,\char`\"{}problem with \char`\"{}\texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<}Get\+Name()\texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<} Gateendl);

will push the 3rd argument in std\+::cout if the message level of \char`\"{}\+Core\char`\"{} messages is greater or equal to 4. which means that it generates a message of level 4 (0 \+: very important/always displayed ... 9 \+: deep debug message).

At run time, one is able to change the level of the messages displayed by using a command like \+:

Gate\+::\+Message\+Manager\+::\+Set\+Message\+Level(\char`\"{}\+Core\char`\"{},5);

which tells the manager to display all Core messages of level up to 5.

Variants \+: Gate$\ast$\+Cont \+: continues a previous Gate\+Message on the same line (without rewriting the type and level) Gate$\ast$\+Inc / Dec \+: displays the message and then increments/decrement the messages tabulation \mbox{[}Debug\mbox{]}Inc\+Tab / \mbox{[}Debug\mbox{]}Dec\+Tab \+: increment/decrement tabulation without dispalying message
