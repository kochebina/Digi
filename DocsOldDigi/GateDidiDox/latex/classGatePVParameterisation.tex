\doxysubsection{Detailed Description}
Base-\/class for GATE physical volume parameterisations. 


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGatePVParameterisation}{Gate\+PVParameterisation}} -\/ by \href{mailto:Daniel.Strul@iphe.unil.ch}{\texttt{ Daniel.\+Strul@iphe.\+unil.\+ch}} \& \href{mailto:Giovani.Santin@cern.ch}{\texttt{ Giovani.\+Santin@cern.\+ch}}
\item This class is meant to allow the use of parameterised volumes in GATE
\item It declares a pure virtual method \mbox{\hyperlink{classGatePVParameterisation_ae8ab4ccac5d3e91259e041f87aea036d}{Get\+Nb\+Of\+Copies()}} to return the number of copies to be made of the volume. This method must be implemented in concrete classes derived from \mbox{\hyperlink{classGatePVParameterisation}{Gate\+PVParameterisation}}.
\item Note that this base-\/class does not implement the pure virtual method Compute\+Transformation() declared by G4\+VPVParameterisation; this method must be implemented in concrete classes derived from \mbox{\hyperlink{classGatePVParameterisation}{Gate\+PVParameterisation}}.
