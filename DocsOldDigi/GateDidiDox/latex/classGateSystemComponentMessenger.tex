\doxysubsection{Detailed Description}
Base class for \mbox{\hyperlink{classGateSystemComponent}{Gate\+System\+Component}} messengers. 


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGateSystemComponentMessenger}{Gate\+System\+Component\+Messenger}} -\/ by \href{mailto:Daniel.Strul@iphe.unil.ch}{\texttt{ Daniel.\+Strul@iphe.\+unil.\+ch}}
\item The \mbox{\hyperlink{classGateSystemComponentMessenger}{Gate\+System\+Component\+Messenger}} inherits from the abilities/responsabilities of the \mbox{\hyperlink{classGateClockDependentMessenger}{Gate\+Clock\+Dependent\+Messenger}} base-\/class, i.\+e. the creation and management of a Gate UI directory for a Gate object
\item In addition, it proposes and manages UI commands that are specific to system components\+: \textquotesingle{}attach\textquotesingle{}, and \textquotesingle{}describe\textquotesingle{}
\item Note\+: from July to Oct. 2002, a system was a vector of system-\/levels, each level managing a vector of components. On Oct. 2002, the whole system (which was too complex) was redesigned, and the current mechanism (tree of system-\/components) replaced the previous one (vector of system-\/levels). The system-\/component messengers are now connected to these new system-\/components rather than to the old ones. 
\end{DoxyItemize}
