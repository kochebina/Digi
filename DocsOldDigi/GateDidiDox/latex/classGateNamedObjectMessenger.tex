\doxysubsection{Detailed Description}
Base class for Gate\+Named\+Volume messengers. 


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGateNamedObjectMessenger}{Gate\+Named\+Object\+Messenger}} -\/ by \href{mailto:Daniel.Strul@iphe.unil.ch}{\texttt{ Daniel.\+Strul@iphe.\+unil.\+ch}}
\item The main function of a \mbox{\hyperlink{classGateNamedObjectMessenger}{Gate\+Named\+Object\+Messenger}} is to create/manage a (new) UI directory for a \mbox{\hyperlink{classGateNamedObjectMessenger}{Gate\+Named\+Object\+Messenger}}
\item It also proposes/handles the UI command \textquotesingle{}describe\textquotesingle{}.
\item Note that, since the \mbox{\hyperlink{classGateNamedObjectMessenger}{Gate\+Named\+Object\+Messenger}} is based on the \mbox{\hyperlink{classGateMessenger}{Gate\+Messenger}} base-\/class, it inherits the methods Gate\+Messenger\+::\+Tell\+Geometry\+To\+Update() and Gate\+Messenger\+::\+Tell\+Geometry\+To\+Rebuild() that allow messengers to command the geometry updating. 
\end{DoxyItemize}
