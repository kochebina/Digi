\doxysubsection{Detailed Description}
Pulse-\/processor modelling a threshold discriminator based on a sigmoidal function. 


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGateSigmoidalThresholder}{Gate\+Sigmoidal\+Thresholder}} -\/ by \href{mailto:Martin.Rey@epfl.ch}{\texttt{ Martin.\+Rey@epfl.\+ch}} (mars 2003)
\item The method Process\+One\+Pulse of this class models a threshold discriminator based on a sigmoidal function (sigmoidal is a symmetrical function included between 0 and 1)\+: \[\sigma(E)=\frac{1}{1+exp\big(\alpha;\frac{E-E_0}{E_0}\big)}\] The operator choose the threshold, the percentage of acceptance for this threshold and the alpha parameter (which is proportionnel of the slope @ symmetrical point Eo); with these parameters and the input pulse energy the function is calculated and if it\textquotesingle{}s bigger than a random number included between 0 and 1, the pulse is copied into the output pulse-\/list. On the contrary, the input pulse is discarded.
