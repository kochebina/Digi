\doxysubsection{Detailed Description}
Pulse-\/processor modelling a pileup (maximum energy wins) of a crystal-\/block. 


\begin{DoxyItemize}
\item The Pileup is parameterised by its \textquotesingle{}depth\textquotesingle{}\+: pulses will be summed up if their volume IDs are identical up to this depth. For instance, the default depth is 1\+: this means that pulses will be considered as taking place in a same block if the first two figures of their volume IDs are identical
\item A second parameter is added \+: the width of the pilup window
\item The class is largely inspired from the \mbox{\hyperlink{classGateReadout}{Gate\+Readout}} class, but is aimed to work by time and not by event.
