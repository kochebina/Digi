Namespace to provide general-\/purpose utility functions to GATE. 


\begin{DoxyItemize}
\item \mbox{\hyperlink{namespaceGateTools}{Gate\+Tools}} -\/ by \href{mailto:Daniel.Strul@iphe.unil.ch}{\texttt{ Daniel.\+Strul@iphe.\+unil.\+ch}}

\begin{DoxyItemize}

\item \mbox{\hyperlink{namespaceGateTools_a32adf8537009afe94b97c872ad936822}{Find\+Gate\+File()}} to find a file in the GATE directories 
\end{DoxyItemize}
\end{DoxyItemize}
