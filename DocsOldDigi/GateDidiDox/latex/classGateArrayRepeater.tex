\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classGateArrayRepeater}{Gate\+Array\+Repeater}} models a repetition of an object in a 3D matrix,. 

a pattern similar to the repetition of crystals in PET scanner blocks


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGateArrayRepeater}{Gate\+Array\+Repeater}} -\/ by \href{mailto:Daniel.Strul@iphe.unil.ch}{\texttt{ Daniel.\+Strul@iphe.\+unil.\+ch}}
\item The array repeater uses five parameters\+:
\begin{DoxyItemize}
\item the numbers of repetitions along X, Y and Z
\end{DoxyItemize}
\item a repetition vector
\item a centering flag Based on these parameters, it repeats an object at regular steps on a rectangular 3D-\/matrix, the step size in each direction being given by the corresponding coordinate of the repeat vector. If the centering flag is on, the matrix is centered on the volume current position. If this flag is off, the first corner copy in the matrix is located at the volume current position. 
\end{DoxyItemize}
