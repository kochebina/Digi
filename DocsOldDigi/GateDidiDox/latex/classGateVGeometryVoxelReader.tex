\doxysubsection{Detailed Description}
This class can read a file with the material info for a matrix of voxels (digital image) 


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGateVGeometryVoxelReader}{Gate\+VGeometry\+Voxel\+Reader}} -\/ by \href{mailto:Giovanni.Santin@cern.ch}{\texttt{ Giovanni.\+Santin@cern.\+ch}}
\item It reads the image file, asks the translator to convert the digital info into material info, and stores the material information. This material info is used by the Gate\+Voxel\+Replica\+Matrix\+Inserter at the moment of the matrix construction
\item The translator has to be inserted befor reading the image file
\item The material information is a generic map to allow for non cubic voxel material distribution (like voxels in a spheric envelope)
