\doxysubsection{Detailed Description}
Namespace to catch and process system signals. 


\begin{DoxyItemize}
\item \mbox{\hyperlink{namespaceGateSignalHandler}{Gate\+Signal\+Handler}} -\/ by \href{mailto:Daniel.Strul@iphe.unil.ch}{\texttt{ Daniel.\+Strul@iphe.\+unil.\+ch}} (June 7 2002)
\item The \mbox{\hyperlink{namespaceGateSignalHandler}{Gate\+Signal\+Handler}} namespace is a collection of routines allowing to catch and process system signals.
\item For now, its only task is to catch SCTRL-\/\textbackslash{} (SIGQUIT) so that the user can stop an acquisition and get back in idle mode.
