\doxysubsection{Detailed Description}
Pulse-\/processor for simulating a quantum efficiency. 


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGateQuantumEfficiency}{Gate\+Quantum\+Efficiency}} -\/ by \href{mailto:Martin.Rey@epfl.ch}{\texttt{ Martin.\+Rey@epfl.\+ch}} (dec 2002)
\item Pulse-\/processor for simulating the quantum efficiency on each channel of a PM or an APD. There are two options\+: one, is to give an unique quantum efficiency for all the channels, and, the other, is to give some lookout tables which contains data and this class takes these data with small variations (minus than 2\%) for creating the tables before the simulation.
