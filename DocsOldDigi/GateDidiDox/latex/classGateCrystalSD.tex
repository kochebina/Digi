\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classGateCrystalSD}{Gate\+Crystal\+SD}} is a sensitive detector , derived from G4\+VSensitive\+Detector,. 

to be attached to one or more volumes of a scanner


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGateVolumeID}{Gate\+Volume\+ID}} -\/ by \href{mailto:Giovanni.Santin@cern.ch}{\texttt{ Giovanni.\+Santin@cern.\+ch}}
\item A \mbox{\hyperlink{classGateCrystalSD}{Gate\+Crystal\+SD}} can be attached to one or more volumes of a scanner. These volumes are essentially meant to be scintillating elements (crystals) but the \mbox{\hyperlink{classGateCrystalSD}{Gate\+Crystal\+SD}} can also be attached to non-\/scintillating elements such as collimators, shields or septa.
\item A \mbox{\hyperlink{classGateCrystalSD}{Gate\+Crystal\+SD}} can be attached only to those volumes that belong to a system (i.\+e. that are connected to an object derived from \mbox{\hyperlink{classGateVSystem}{Gate\+VSystem}}). Once a \mbox{\hyperlink{classGateCrystalSD}{Gate\+Crystal\+SD}} has been attached to a volume that belongs to a given system, it is considered as attached to this system, and can be attached only to volumes that belong to the same system.
\item The \mbox{\hyperlink{classGateCrystalSD}{Gate\+Crystal\+SD}} generates hits of the class \mbox{\hyperlink{classGateCrystalHit}{Gate\+Crystal\+Hit}}, which are stored in a regular hit collection. 
\end{DoxyItemize}
