\doxysubsection{Detailed Description}
Pulse-\/processor for simulating an optical and/or an electronic crosstalk. 


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGateCrosstalk}{Gate\+Crosstalk}} -\/ by \href{mailto:Martin.Rey@epfl.ch}{\texttt{ Martin.\+Rey@epfl.\+ch}} (dec 2002)
\item Pulse-\/processor for simulating an optical and/or an electronic crosstalk of the scintillation light between the neighbor crystals\+: if the input pulse arrives in a crystal array, pulses around it are created (in the edge and corner neighbor crystals). ATTENTION\+: this module functions only for a chosen volume which is an array repeater !!!
