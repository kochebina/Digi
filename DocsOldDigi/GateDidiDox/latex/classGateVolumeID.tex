\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classGateVolumeID}{Gate\+Volume\+ID}} stores all the navigation information needed for reaching and. 

identifying a specific real-\/space volume (i.\+e. a touchable)


\begin{DoxyItemize}
\item \mbox{\hyperlink{classGateVolumeID}{Gate\+Volume\+ID}} -\/ by \href{mailto:Daniel.Strul@iphe.unil.ch}{\texttt{ Daniel.\+Strul@iphe.\+unil.\+ch}} (May 7 2002)
\item A \mbox{\hyperlink{classGateVolumeID}{Gate\+Volume\+ID}} stores a navigation path to a specific real-\/space volume as a vector of \mbox{\hyperlink{classGateVolumeSelector}{Gate\+Volume\+Selector}}.
\item Once a \mbox{\hyperlink{classGateVolumeID}{Gate\+Volume\+ID}} has been created for a touchable, it can provide the following services\+:
\begin{DoxyItemize}
\item Get any volume of the path (i.\+e. the touchable itself or any of its ancestors)
\item Transfer a position from a volume\textquotesingle{}s reference frame into another volume\textquotesingle{}s reference frame
\item Return a \char`\"{}daughter\+ID\char`\"{} for each level 
\end{DoxyItemize}
\end{DoxyItemize}
