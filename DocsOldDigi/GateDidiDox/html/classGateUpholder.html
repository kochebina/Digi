Detailed Description

Pulse-processor modelling a simple uphold limit.

  • GateUpholder - by Steve.nosp@m.n.St.nosp@m.aelen.nosp@m.s@ru.nosp@m.g.ac..nosp@m.be

  • The method ProcessOnePulse of this class models a simple uphold limit: any input pulse whose energy is below the energy limit is copied into the output pulse-list. On the contrary, any input pulse whose energy is above this limit is discarded.
