Detailed Description

Provides some basic fonctionalities for GATE messengers.

  • GateMessenger - by Danie.nosp@m.l.St.nosp@m.rul@i.nosp@m.phe..nosp@m.unil..nosp@m.ch
  • The main function of a GateMessenger is to create (as needed) and manage a UI messenger directory.
  • It stores the directory base-name and directory-name ( = "/gate/" + base-name), and a pointer to the directory
  • It also provides the ability to communicate with the geometry, and to ask it to update or rebuild itself (OK... storing this method here is not very OO...)
