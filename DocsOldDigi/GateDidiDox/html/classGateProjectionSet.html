Detailed Description

Structure to store the projection sets from a SPECT simulation.

  • GateProjectionSet - by Danie.nosp@m.l.St.nosp@m.rul@i.nosp@m.phe..nosp@m.unil..nosp@m.ch (Oct. 2002)
  • This structure is generated during a SPECT simulation by GateToProjectionSet. It can be stored into an output file using a set-writer such as GateToInterfile
