Detailed Description

Pulse-processor for simulating an optical and/or an electronic crosstalk.

  • GateCrosstalk - by Marti.nosp@m.n.Re.nosp@m.y@epf.nosp@m.l.ch (dec 2002)

  • Pulse-processor for simulating an optical and/or an electronic crosstalk of the scintillation light between the neighbor crystals: if the input pulse arrives in a crystal array, pulses around it are created (in the edge and corner neighbor crystals). ATTENTION: this module functions only for a chosen volume which is an array repeater !!!
