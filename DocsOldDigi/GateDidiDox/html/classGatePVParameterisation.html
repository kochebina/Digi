Detailed Description

Base-class for GATE physical volume parameterisations.

  • GatePVParameterisation - by Danie.nosp@m.l.St.nosp@m.rul@i.nosp@m.phe..nosp@m.unil..nosp@m.ch & Giova.nosp@m.ni.S.nosp@m.antin.nosp@m.@cer.nosp@m.n.ch
  • This class is meant to allow the use of parameterised volumes in GATE
  • It declares a pure virtual method GetNbOfCopies() to return the number of copies to be made of the volume. This method must be implemented in concrete classes derived from GatePVParameterisation.

  • Note that this base-class does not implement the pure virtual method ComputeTransformation() declared by G4VPVParameterisation; this method must be implemented in concrete classes derived from GatePVParameterisation.
