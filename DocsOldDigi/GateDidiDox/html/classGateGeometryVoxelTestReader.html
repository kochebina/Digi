Detailed Description

This class is a concrete implementation of the abstract GateVGeometryVoxelReader.

  • GateGeometryVoxelTestReader - by Giova.nosp@m.nni..nosp@m.Santi.nosp@m.n@ce.nosp@m.rn.ch

  • it reads a the voxel info from an ASCII file with the sequence of info:

    • nx ny nz
    • dx dy dz (mm)
    • number_of_listed_voxels
    • i1 j1 k1 materialName1
    • i2 j2 k2 materialName2
    • ...
