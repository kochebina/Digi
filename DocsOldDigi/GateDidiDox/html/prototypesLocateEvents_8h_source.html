21 Description of prototypesLocateEvents.h:
22
23 Functions used for the calculation of events position
24 in the 3D laboratory (x.y.z) system:
25 ->locateEventInLaboratory - Compute the elements 3D coordinates
26 in the laboratory (x,y,z) system
27 ->fillInStructScannerGeometry - Initialize the members of the structure
28 LMF_cch_scannerGeometry
29 ->setLayersInfo - Initialize the radial size and the interaction length in each layer
30 ->setSubstructuresValues - Create an array ("substructureNumericalValues"),
31 which contains the substructures numerical values
32 ->testPitchVersusSize - Control the value of the substructures pitches in comparison
33 with the substructures sizes
34 ->testAngleDefaultUnit - Test if the angle default unit is the radian
35 ->locateID - Define the crystal or the submodule or the module or the rsector tangential and axial ID
36 ->locateSubstructureInStructure - Compute the 3D coordinates of substructure(id_t,id_z)
37 in a tangential reference frame attached to a rsector
38 ->locateRsectorInLaboratory - Process to calculate the 3D coordinates of a rsector in the laboratory (x,y,z) system
