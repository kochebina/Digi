Detailed Description

Class for storing a 'pulse' (luminous or electronic) derived from one or more hits.

  • GatePulse - by Danie.nosp@m.l.St.nosp@m.rul@i.nosp@m.phe..nosp@m.unil..nosp@m.ch
  • When hits are registered, theyr are converted into pulses by a GateHitConvertor
  • These pulses may then be processed by a series of pulse-processors. The output of this processing is a pulse-list, which is then converted into digis.

  • S. Stute: june2014, add two methods used in the new GateReadout implementation

    See also
    GateVPulseProcessor, GatePulseProcessorChain
