Detailed Description

Structure to store the sinogram sets from a PET simulation.

  • GateSinogram - by By jan@s.nosp@m.hfj..nosp@m.cea.f.nosp@m.r & comta.nosp@m.t@sh.nosp@m.fj.ce.nosp@m.a.fr (december 2002)
  • This structure is generated during a PET simulation by GateToSinogram. It can be stored into an output file using a set-writer such as GateSinoToEcat7
