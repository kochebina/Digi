Detailed Description

Base class for GateCylindricalPETSystem messengers.

  • GateCylindricalPETSystemMessenger - by Danie.nosp@m.l.St.nosp@m.rul@i.nosp@m.phe..nosp@m.unil..nosp@m.ch
  • The GateCylindricalPETSystemMessenger inherits from the abilities/responsabilities of the GateClockDependentMessenger base-class: creation and management of a Gate UI directory for a Gate object; creation of the UI command "describe"
  • In addition, it proposes and manages the UI commands 'enable' and 'disable'.
