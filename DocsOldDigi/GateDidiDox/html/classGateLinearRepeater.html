Detailed Description

The GateLinearRepeater models a repetition of an object along a line,.

a pattern similar to the repetition of the scanner rings in a PET scanner

  • GateLinearRepeater - by Danie.nosp@m.l.St.nosp@m.rul@i.nosp@m.phe..nosp@m.unil..nosp@m.ch
  • The linear repeater uses three parameters:
    • a number of repetitions
  • a repetition vector
  • a centering flag Based on these parameters, it repeats an object at regular steps in a line, the step size being given by the repeat vector. If the centering flag is on, the series of copies is centered on the volume current position. If this flag is off, the first copy in the series is located at the volume current position.
