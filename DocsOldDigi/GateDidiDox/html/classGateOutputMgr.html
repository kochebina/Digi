GateOutputMgr::GateOutputMgr ( const G4String  name)
protected

Private constructor (the class is a singleton)

In the constructor we build the list of the output modules to attach to the Mgr. In the future this will be substituted by a dynamic insertion of modules of predefined type. A Messenger of the type GateOutputMgrMessenger is built.

Parameters
nameThe name of the Output Manager is necessary to build the commands directory in the Messenger
