Detailed Description

A messenger for a GateSystemListManager.

  • GateSystemListManagerMessenger - by Danie.nosp@m.l.St.nosp@m.rul@i.nosp@m.phe..nosp@m.unil..nosp@m.ch
  • The GateSystemListManagerMessenger inherits from the abilities/responsabilities of the GateClockDependentMessenger base-class, i.e. the creation and management of a Gate UI directory for a Gate object
  • In addition, the main responsability of this messenger is to handle the system store, and to allow the insertion of new systems.
  • It proposes and manages commands specific to the system-stores (these commands are fairly similar to those of other insertion messengers): definition of the name of a new system, listing of system choices, list of already created systems, and insertion of a new system
