Detailed Description

Pulse-processor modelling a calibration Nphotons->Energy.

  • GateCalibration - by Marti.nosp@m.n.Re.nosp@m.y@epf.nosp@m.l.ch (feb 2003)

  • Can be usefull if you use the class(es) GateLightYield, GateTransferEfficiency or GateQuantumEfficiency. You can also set your own calibration factor. Allows the "recalibration" in energy.
