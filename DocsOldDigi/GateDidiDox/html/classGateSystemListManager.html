Detailed Description

Stores the list of systems.

  • GateSystemListManager - by Danie.nosp@m.l.St.nosp@m.rul@i.nosp@m.phe..nosp@m.unil..nosp@m.ch (May 16 2002)

  • The GateSystemListManager is a singleton. Its task is to handle a list of systems, provide tools for retrieving systems ort system-components, and allow the insertion of new systems.
