Detailed Description

Class for storing an identifier of a volume as a vector of integer.

This volume ID can be stored into output files for analysis and image reconstruction

  • GateOutputVolumeID - by Danie.nosp@m.l.St.nosp@m.rul@i.nosp@m.phe..nosp@m.unil..nosp@m.ch

  • Output volume IDs are created by systems (GateVSystem) after analysis of a hit: Each output volume ID identifies a volume according to the geometry model defined by the system
