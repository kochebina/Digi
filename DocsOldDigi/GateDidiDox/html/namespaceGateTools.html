Detailed Description

Namespace to provide general-purpose utility functions to GATE.

  • GateTools - by Danie.nosp@m.l.St.nosp@m.rul@i.nosp@m.phe..nosp@m.unil..nosp@m.ch
  • The GateTools namespace is a collection of functions providing some general-purpose utility functions to GATE:
    • Indent() to compute a string made of tabulations
    • GetBaseName() to extract the beginning of a string terminated by a tag
    • FindGateFile() to find a file in the GATE directories
