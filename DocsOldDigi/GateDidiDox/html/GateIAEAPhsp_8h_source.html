54/************************************************************************
55* Initialization
56*
57* Given a file name header_file of length hf_length, initialize a
58* new IAEA particle source, assign an unique source_ID to it and return
59* this Id in result. Don’t assume header_file is null-terminated as the
60* function may be called from a Fortran program.
61* The need for an Id arises from the fact that some applications may
62* want to use several IAEA sources at once. The implementation must therefore
63* maintain a list of already initialized sources.
64* If an error occures (e.g. header file does not exist, there are errors
65* in the header file, etc.), assign a negative number to result.
66* (one may want to specify a list of error codes so that the application
67* knows what went wrong). This function *must* be called before using
68* any of the following functions for a given source id.
69*
70* access = 1 => opening read-only file
71* access = 2 => opening file for writing
72* access = 3 => opening file for appending/updating
73*
74***********************************************************************/
75IAEA_EXTERN_C IAEA_EXPORT
76void iaea_new_source(IAEA_I32 *source_ID, char *header_file,
77 const IAEA_I32 *access, IAEA_I32 *result,
78 int hf_length);

126/*******************************************************************************
127* Set a type type of the extra long variable corresponding to the "index" number
128* for a corresponding header of the phsp "id". Index is running from zero.
129*
130* The current list of types for extra long variables is:
131* 0: User defined generic type
132* 1: Incremental history number (EGS,PENELOPE)
133* = 0 indicates a nonprimary particle event
134* > 0 indicates a primary particle. The value is equal to the number of
135* primaries particles employed to get to this history after the last
136* primary event was recorded.
137* 2: LATCH (EGS)
138* 3: ILB (PENELOPE)
139* more to be defined
140*
141* Usually called before writing phsp header to set the type of extra long
142* variables to be stored. It must be called once for every extralong variable.
143*
144* type = -1 means the source's header file does not exist
145* or source was not properly initialized (call iaea_new_...)
146* type = -2 means the index is out of range ( 0 <= index < NUM_EXTRA_LONG )
147* type = -3 means the type to be set is out of range
148* ( 1 <= type < MAX_NUMB_EXTRALONG_TYPES )
149*******************************************************************************/
150IAEA_EXTERN_C IAEA_EXPORT
151void iaea_set_type_extralong_variable(const IAEA_I32 *id,
152 const IAEA_I32 *index,
153 IAEA_I32 *type);

285/**************************************************************************
286* Partitioning for parallel runs
287*
288* i_parallel is the job number, i_chunk the calculation chunk,
289* n_chunk the total number of calculation chunks. This function
290* should divide the available phase space of source with Id id
291* into n_chunk equal portions and from now on deliver particles
292* from the i_chunk’th portion. (i_chunk must be between 1 and n_chunk)
293* The extra parameter i_parallel is needed
294* for the cases where the source is an event generator and should
295* be used to adjust the random number sequence.
296* The variable is_ok should be set to 0 if everything went smoothly,
297* or to some error code if it didn’t.

326/**************************************************************************

328*
329* Return the next particle from the sequence of particles from source
330* with Id id. Set n_stat to the number of statistically independent
331* events since the last call to this function (i.e. n_stat = 0, if
332* the particle resulted from the same incident electron, n_stat = 377
333* if there were 377 statistically independent events sinc the last particle
334* returned, etc.). If this information is not available,
335* simply set n_stat to 1 if the particle belongs to a new statistically
336* independent event. Set n_stat to -1, if a source with Id id does not
337* exist. Set n_stat to -2, if end of file of the phase space source reached
338**************************************************************************/
