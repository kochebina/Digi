Detailed Description

Pulse-processor for simulating a quantum efficiency.

  • GateQuantumEfficiency - by Marti.nosp@m.n.Re.nosp@m.y@epf.nosp@m.l.ch (dec 2002)

  • Pulse-processor for simulating the quantum efficiency on each channel of a PM or an APD. There are two options: one, is to give an unique quantum efficiency for all the channels, and, the other, is to give some lookout tables which contains data and this class takes these data with small variations (minus than 2%) for creating the tables before the simulation.
