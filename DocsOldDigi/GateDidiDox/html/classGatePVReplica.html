Detailed Description

Base-class for replicas used in the GATE geometry.

  • GatePVReplica - by Danie.nosp@m.l.St.nosp@m.rul@i.nosp@m.phe..nosp@m.unil..nosp@m.ch
  • This class is an extension of the regular class G4PVReplica.
  • Its major feature is that it provides a public method Update() so that the replica can be updated rather than rebuilt.
  • Only replication along the cartesian axes is allowed
