Detailed Description

Extension of the G4RunManager to provide new functions.

and automated resetting of the navigator at the beginning of each run

  • GateRunManager - by Danie.nosp@m.l.St.nosp@m.rul@i.nosp@m.phe..nosp@m.unil..nosp@m.ch
  • A GateRunManager is derived from G4RunManager, and provides the following functionalities:
  • PhysicsHasChanged(): to reset the physicsInitialised flag
  • InitGeometryOnly(): new method to initialise only the geometry, allowing us to build the GATE geometry
  • RunInitialisation(): overload of G4RunManager()::RunInitialisation() that resets the geometry navigator.
