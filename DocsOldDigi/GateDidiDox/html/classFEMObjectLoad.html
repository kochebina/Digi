Detailed Description

Define a FE Mesh Load - This is a general purpose container able to hold information for any Load Type. The Load contains the following items: 1) Global Model number 2) Load Class Name (i.e. Type) 3) Element global Number on which load is applied 4) Number of dimensions 5) Force vector 6) Degrees of freedom 7) Node Number 8) Number of Right Hand Side components 9) Right Hand Side 10) Number of Left Hand Side components 11) Left Hand Side 12) Number of Elements 13) Element Ids 14) Force Matrix 15) Edge Number 16) Undeformed point (for landmark load) 17) Deformed point (for landmark load) 18) Variance
