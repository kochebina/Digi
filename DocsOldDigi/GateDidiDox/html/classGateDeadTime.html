Detailed Description

Pulse-processor modelling a simple dead time discriminator.

  • GateDeadTime - by Luc.S.nosp@m.imon.nosp@m.@iphe.nosp@m..uni.nosp@m.l.ch
  • The method ProcessOnePulse of this class models a simple deadTime discriminator. User chooses value of dead time, mode (paralysable or not) and geometric level of application (crystal, module,...)
