Detailed Description

Pulse-processor for simulating the transfer efficiency on each crystal.

  • GateTransferEfficiency - by Marti.nosp@m.n.Re.nosp@m.y@epf.nosp@m.l.ch (jan 2003)

  • Allows to specify a transfer coefficient for each type of crystals.
